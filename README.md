# Appointment Booking API

Pet-project на FastAPI с системой записи.

## 🔥 Функционал

- JWT авторизация
- Роли (client / master)
- Услуги мастеров
- Бронирование
- Проверка пересечений
- Подтверждение записей

## ⚙️ Стек

- FastAPI
- SQLModel
- SQLite
- JWT
- HTML/CSS/JS frontend
